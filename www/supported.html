Supported Databases

Most major databases are currently supported by ActiveObjects in release 0.6. This data is only current to the 0.6 stable release.

Database URI Protocol Support
Derby jdbc:derby
jdbc:derby:// 		Well tested and stable
HSQLDB jdbc:hsqldb
jdbc:hsqldb:// 		Stable
MS SQL Server 2005 and 2007
(Microsoft and JTDS drivers) 		jdbc:sqlserver://
jdbc:jtds:sqlserver:// 		Stable
MySQL jdbc:mysql:// Well tested and very stable
Oracle jdbc:oracle:thin
jdbc:oracle:oci 		Stable, but not well tested
PostgreSQL jdbc:postgresql:// Stable
