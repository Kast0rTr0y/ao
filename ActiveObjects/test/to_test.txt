This is a comprehensive list of things which can and should be unit tested and are not currently.

 * Generator generate: spot-check DDL
 
 * SchemaReader readSchema: spot-check DDL
 * SchemaReader diffSchema: compare identical generated schemas
 * SchemaReader sortTopologically: construct unordered DDL, spot-check ordering
 
 * TypeManager get type for class: Map<Class<?>, DatabaseType> results including custom types
 * TypeManager get type for int: Map<Class<?>, int> results
 
 * DatabaseType#getType
 * DatabaseType#getDefaultPrecision
 * DatabaseType#isHandlerFor (class and int)
 * DatabaseType#putToDatabase
 * DatabaseType#getDefaultName
 * DatabaseType#convert
 * DatabaseType#defaultParseValue
 
====Perform Against All Databases====
 
 * Query: generate SQL

 * Database schema render
     * Multiple built-in types
     * At least one custom type
     * SQLType
     * Table
     
     * Auto-increment
     * Default
     * Ignore
     * NotNull
     * PrimaryKey
     * Unique
